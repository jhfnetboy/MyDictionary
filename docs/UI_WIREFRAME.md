# MyDictionary - UI 设计稿

## 一、划词翻译浮窗 (Tooltip)

### 1.1 触发态
```
用户选中文本:
  "artificial intelligence"
              ↓
         [🌐 翻译]  ← 小图标按钮浮现
```

### 1.2 展开态
```
┌─────────────────────────────┐
│  artificial intelligence    │ ← 原文(灰色小字)
│  人工智能                    │ ← 翻译结果(黑色大字)
│                             │
│         [详情 →]             │ ← 跳转到侧边栏按钮
└─────────────────────────────┘
  ▼                             ← 小三角指向选中文本
```

**样式规范**:
- 宽度: 200-300px (自适应内容)
- 背景: 白色 `#FFFFFF`
- 边框: 1px 灰色 `#E0E0E0`
- 阴影: `0 2px 8px rgba(0,0,0,0.1)`
- 圆角: `8px`
- 字体:
  - 原文: 12px, `#666666`
  - 翻译: 16px, `#000000`, font-weight: 500

## 六、色彩规范

### 6.1 主题色
- **主色调**: 蓝色 `#1976D2` (Google 蓝)
- **强调色**: 橙色 `#FF9800` (用于按钮 hover)
- **成功色**: 绿色 `#4CAF50`
- **警告色**: 黄色 `#FFC107`
- **错误色**: 红色 `#F44336`

### 6.2 中性色
- **背景色**: 白色 `#FFFFFF`
- **次级背景**: 浅灰 `#F5F5F5`
- **边框色**: 灰色 `#E0E0E0`
- **文字主色**: 黑色 `#000000`
- **文字次色**: 灰色 `#666666`
- **占位符**: 浅灰 `#9E9E9E`

### 6.3 深色模式 (可选)
- **背景色**: 深灰 `#1E1E1E`
- **卡片背景**: 暗灰 `#2D2D2D`
- **文字主色**: 白色 `#FFFFFF`
- **文字次色**: 浅灰 `#B0B0B0`

## 七、动画效果

### 7.1 侧边栏滑入
```css
/* 进入动画 */
@keyframes slideInRight {
  from {
    transform: translateX(100%);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}

/* 退出动画 */
@keyframes slideOutRight {
  from {
    transform: translateX(0);
    opacity: 1;
  }
  to {
    transform: translateX(100%);
    opacity: 0;
  }
}
```

### 7.2 浮窗淡入淡出
```css
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

@keyframes fadeOut {
  from { opacity: 1; }
  to { opacity: 0; }
}
```

### 7.3 骨架屏闪烁
```css
@keyframes shimmer {
  0% { background-position: -468px 0; }
  100% { background-position: 468px 0; }
}
```

## 八、响应式设计

### 8.1 小屏幕适配 (宽度 < 600px)
- 侧边栏宽度调整为 `100vw` (全屏)
- 字体大小缩小 10%
- 按钮区域调整为垂直堆叠

### 8.2 触摸设备优化
- 按钮最小点击区域: 44x44px
- 增加手势支持:
  - 向右滑动关闭侧边栏
  - 双指缩放文本

## 九、无障碍设计 (A11y)

### 9.1 键盘导航
```
Tab        → 切换焦点到下一个可交互元素
Shift+Tab  → 切换焦点到上一个可交互元素
Enter      → 激活当前焦点元素
Esc        → 关闭侧边栏/浮窗
```

### 9.2 屏幕阅读器支持
- 所有按钮添加 `aria-label` 属性
- 翻译结果区域添加 `role="region"` 和 `aria-live="polite"`
- 加载状态添加 `aria-busy="true"`

### 9.3 高对比度模式
- 所有文本对比度 ≥ 4.5:1 (WCAG AA 标准)
- 焦点指示器: 2px 蓝色描边 `#1976D2`
